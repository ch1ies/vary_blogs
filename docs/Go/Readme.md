## GO的常用命令

```
 	1. go build [ 文件名 ]: 编译成.exe结尾的可执行文件
 	2. go run [文件名]: 在内存中运行，不会生成可执行文件
 	3. go install :表示安装的意思，它先编译源代码得到可执行文件，然后将可执行文件移动到`GOPATH`的bin目录下。因为我们的环境变量中配置了`GOPATH`下的bin目录，所以我们就可以在任意地方直接执行可执行文件了。
```
